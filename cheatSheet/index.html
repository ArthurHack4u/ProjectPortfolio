Una guía de referencia rápida con las etiquetas HTML más importantes, explicadas una por una.

Estructura del Documento

<!DOCTYPE html>

No es una etiqueta, sino una declaración. Le dice al navegador que el documento es de tipo HTML5. Es la primera línea obligatoria en cualquier archivo HTML.

<html>

Es el elemento raíz de la página. Todas las demás etiquetas deben estar contenidas dentro de esta.

<head>

Contiene "metadatos" sobre la página, como el título, enlaces a hojas de estilo (CSS), scripts y el set de caracteres. Su contenido no es visible en la página.

<title>

Define el título del documento, que se muestra en la pestaña o en la barra de título del navegador. Se coloca dentro de <head>.

<body>

Contiene todo el contenido visible de la página web: textos, imágenes, videos, formularios, etc.

Secciones Semánticas

<header>

Representa un grupo de contenido introductorio o de navegación. Puede contener encabezados, logos, formularios de búsqueda, etc.

<nav>

Define una sección con los enlaces de navegación principales de la página (menú).

<main>

Especifica el contenido principal y único del documento. Solo debe haber un <main> por página.

<aside>

Define contenido que está indirectamente relacionado con el contenido principal, como una barra lateral (sidebar).

<footer>

Define el pie de página de un documento o sección. Usualmente contiene información del autor, copyright y enlaces de contacto.

Texto y Contenedores

<h1> a <h6>

Definen los encabezados o títulos. <h1> es el de mayor importancia y <h6> el de menor.

<p>

Define un párrafo de texto.

<a>

Define un hipervínculo (ancla o link). Su atributo principal es href, que especifica la URL de destino.

<a href="https://kortex.com">Visita Kortex</a>

<div>

Es un contenedor genérico de bloque. Se usa para agrupar otros elementos y aplicarles estilos con CSS o manipularlos con JavaScript.

<span>

Es un contenedor genérico en línea. Se usa para aplicar estilos a una pequeña parte de un texto dentro de otro elemento, como un párrafo.

Listas

<ul> y <ol>

<ul> (Unordered List) crea una lista con viñetas. <ol> (Ordered List) crea una lista numerada.

<li>

Define un elemento de lista (List Item). Se usa dentro de <ul> y <ol>.

<dl>, <dt> y <dd>

Crean una lista de descripción. <dl> es el contenedor, <dt> es el término a definir y <dd> es la descripción de ese término.

Tablas

<table>

Define el contenedor principal para crear una tabla de datos.

<tr>, <th> y <td>

<tr> (Table Row) define una fila. <th> (Table Header) define una celda de encabezado. <td> (Table Data) define una celda de datos normal.

<thead>, <tbody> y <tfoot>

Agrupan semánticamente las filas de la cabecera, el cuerpo y el pie de la tabla, respectivamente.

Multimedia

<img>

Inserta una imagen. Es una etiqueta auto-cerrada y requiere los atributos src (la ruta de la imagen) y alt (texto alternativo).

<video> y <audio>

Incrustan contenido de video y audio, respectivamente. Usualmente se usa el atributo controls para mostrar los controles de reproducción.

<source>

Permite especificar múltiples formatos de un archivo multimedia (video/audio) para que el navegador elija el más compatible. Se usa dentro de <video> o <audio>.

Formularios

<form>

Es el contenedor para todos los controles de un formulario que enviará datos a un servidor.

<input>

El elemento de formulario más versátil. Su comportamiento cambia drásticamente según su atributo type (ej: text, password, checkbox, radio, submit, etc.).

<label>

Define una etiqueta de texto para un elemento <input>. Mejora la accesibilidad al asociarse con el control del formulario.

<select> y <option>

<select> crea un menú desplegable. Cada <option> dentro del select es una de las opciones disponibles en la lista.

<textarea>

Define un campo de entrada de texto de múltiples líneas.

Consejo Rápido

Usa siempre etiquetas semánticas. Ayuda a los lectores de pantalla y a los motores de búsqueda a entender mejor tu contenido.
