Una guía de referencia rápida con las etiquetas más importantes

Estructura del Documento

<!DOCTYPE html>, <html>, <head>, <body>

Definen la estructura fundamental de una página. <head> contiene metadatos y <body> el contenido visible.

Secciones Semánticas

<header>, <nav>, <main>, <aside>, <footer>

Organizan el contenido de la página de forma lógica, mejorando la accesibilidad y el SEO.

Texto y Contenedores

<h1> - <h6>, <p>, <span>, <div>

Elementos para títulos, párrafos y contenedores genéricos de bloque (div) o en línea (span).

<a>

Crea un hipervínculo (link). El atributo href especifica el destino.

<a href="https://google.com">Ir a Google</a>

Listas

<ul>, <ol>, <li>, <dl>, <dt>, <dd>

Para crear listas no ordenadas (ul), ordenadas (ol) y de descripción (dl).

Consejo Rápido

Usa siempre etiquetas semánticas. Ayuda a los lectores de pantalla y a los motores de búsqueda a entender mejor tu contenido.
